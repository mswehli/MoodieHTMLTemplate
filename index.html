Grid System

Grid system uses 3 basic mixins
  • grid-container($unitSize:4,$xunits:"", $yunits:"",$maxWidth:100%) : Used for main parent container of the grid system. There should only be and they should not be parented from each other
  • grid-subgrid($unitSize:2, $xunits:4, $yunits:4): a grid within a grid. There can be several and they can be parented by each other or grid-containers but not by grid-items
  • grid-item($xunits:"") : individual grid box. These are the actual containers for items within the grid. There can be many and can be parented by grid-containers and grid-subgrids but not by themselves (grid-items)
